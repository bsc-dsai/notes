Introduction to Statistics - Syllabus

Course Outline

This course will provide a thorough understanding of the following topics over 12 weeks:

Week 1-2.5: Basics of Probability
  • Introduction to Probability
  • Relative Frequency Definition of Probability
  • Sample Space
  • Categorizing Sample Spaces
  • Set Operations and Properties
  • Axioms of probability
  • Property: Complement Rule of Probability
  • Property: P(Φ) = 0
  • Property: P(A ∪ B) for Non-Mutually Exclusive Sets
  • Property: P(A) ≤ P(B) if A ⊆ B
Week 3: Introduction to Random Variables
  • Understanding random variables
  • Types of random variables
Week 4-5: Characterizing Random Variables
  • Cumulative Distribution Function (CDF)
  • Probability Mass Function (PMF)
  • Probability Density Function (PDF)
Week 6: Standard Discrete and Continuous Distributions
  • Commonly encountered random variables
  • Discrete and continuous distributions
Week 6-8: Expectation, Variance, and Moment Generating Functions
  • Expectation and variance of random variables
  • Moment generating functions
Week 9: Functions of Random Variables
  • Transforming one random variable to another
  • Characterizing the transformed variable
Week 10: Application of Bayes' Rule
  • Applying Bayes' rule for random variables
Week 11-12: Two-Dimensional Random Variables
  • Joint PDF and marginal PDF
  • Conditional distributions
  • Covariance and correlation

Assessment Structure

Each week will include:

  • Video lectures
  • Live sessions
  • Ungraded practice assessments
  • Graded assessments (quizzes)
